# The Library

The Library is a playground application where I am exploring different concepts in application development.
This time, it's written in PHP 8.1, using Laravel.

While its true *purpose* is that of learning and exploration, the *premise* is to create a functional
multi-user web application that serves as central hub for cataloguing, trading and lending of personal
'Libraries';

The intent is to provide a way for individuals to easily represent their own collections of different media
and allow personal collections to be easily searchable. Users will be able to choose whether all, some or none
of their collection is made public, whether just for bragging rights or for the purpose of trading and/or loaning
with other users.
